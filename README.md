Reportly is a production-style multi-agent AI system that automatically generates structured reports from a single topic using an Orchestrator–Worker architecture powered by LangGraph. The system exposes a REST API using FastAPI and is fully containerized using Docker for reproducible deployment.

This project demonstrates modern AI engineering patterns including multi-agent orchestration, parallel task execution, structured outputs, and containerized services.

**Key Features**
Multi-agent workflow using LangGraph
Orchestrator–Worker architecture
Parallel section generation
Structured output with Pydantic schemas
FastAPI REST API endpoint
Dockerized deployment
Environment-based configuration
